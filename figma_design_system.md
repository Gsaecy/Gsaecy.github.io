# AI智汇观察 - Figma设计系统

## 🎯 设计原则

### 1. 品牌一致性
- **主色调**：科技蓝 (#2962ff) + 深蓝 (#1a237e) + 亮橙 (#ff6d00)
- **辅助色**：浅灰 (#f5f5f5) + 深灰 (#424242) + 白色 (#ffffff)
- **字体**：思源黑体（中文） + SF Pro Display（英文）

### 2. 内容层次
- **一级标题**：24-32px，加粗，主色
- **二级标题**：18-24px，加粗，深色
- **正文**：14-16px，常规，深灰
- **标注**：12-14px，常规，浅灰

### 3. 间距系统
- **大间距**：32px（区块之间）
- **中间距**：16px（元素之间）
- **小间距**：8px（紧密元素）

## 🔧 实现步骤

### 第一步：创建Figma设计文件
1. 打开你的 `AI-Blog-Templates` 文件
2. 按照上述规范设计三个页面
3. 使用组件（Components）确保一致性

### 第二步：标记关键节点
为以下元素设置明确的命名和ID：
1. **标题区域**：`title`, `subtitle`
2. **正文区域**：`content`, `key_points`
3. **数据区域**：`data_card_1`, `data_card_2`
4. **CTA区域**：`cta`, `qrcode`
5. **元信息**：`date`, `author`, `tags`

### 第三步：测试自动化
1. 运行节点ID获取脚本
2. 更新配置文件
3. 测试内容填充和导出
